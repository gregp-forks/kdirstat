Documentation for KTreeMap


KTreeMap is a Qt/KDE widget written for KDirStat that displays information about disk/directory usage as a Treemap graphics.

KTreeMap is written by Alexander Rawass <alexannika@users.sourceforge.net>
KDirStat is written by Stefan Hundhammer <sh@suse.de>

KTreeMap is currently in heavy development, expect newer versions any time
